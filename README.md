## 运行方法


### 方法1 运行python文件
- 下载或克隆项目到本地一个空文件夹（建议）
- 进入项目目录，运行`python main.py`
### 方法2 运行exe文件
- 下载或克隆项目到本地一个空文件夹（建议）
- 进入项目目录，双击运行`main.exe`
